### ass1-skel.c
ass1-skel.c is the code for 2023 FOA assignment 1. <br>
In this project we developed a high-precision numeric calculator by using C arrays to store the digits of the numbers, 
so it won’t quite be arbitrary precision.

### ass2-skel.c
ass2-skel.c is the code for 2023 FOA assignment 2. <br>
In this project, we designed a tool for generating text statements from prompts. This automaton is built based on dynamic memory and linked data structures. The tool generated the most likely extension of the prompt, that is, the extension that follows the most frequent subsequent nodes.
